# Contribuir

Gracias por tu interés en mejorar este proyecto.

## Flujo
1. Fork y rama: `feature/mi-cambio` o `fix/mi-arreglo`.
2. Cambios enfocados (un tema por PR).
3. Verificá que compile / pase la CI.
4. Conventional Commits (`feat:`, `fix:`, `docs:`, `chore:`).
5. Abrí un Pull Request describiendo qué cambia y por qué.
